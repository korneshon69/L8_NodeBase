# Лабораторная работа: L8_NodeBase 🚀

## 📌 Особенности проекта
* **Управление окружением:** Динамическое переключение между `development`, `production` и `domain` через `.env` файлы.
* **Безопасность:** Хеширование паролей с использованием `bcrypt`.
* **Файловая система:** Реализация синхронных и асинхронных методов работы с файлами и папками (без использования потоков).
* **Автоматизация:** Использование `nodemon` для автоперезапуска при разработке.
* **Интеграция с API:** Загрузка данных с JSONPlaceholder и их обработка.
